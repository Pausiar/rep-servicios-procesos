# PSP Recovery Lab

Web estatica para estudiar la recuperacion de PSP de Pau Silvestre.

## Abrir en local

Abre `index.html` en el navegador.

## Publicar en GitHub Pages

1. Sube esta carpeta a un repositorio de GitHub.
2. En GitHub, entra en `Settings` -> `Pages`.
3. En `Build and deployment`, elige `Deploy from a branch`.
4. Selecciona la rama `main` y la carpeta `/root`.
5. Guarda. GitHub publicara la web en unos minutos.

## Temas incluidos

- Streams
- Method Reference
- AtomicReference y variables atomicas
- Concurrent Collections
- ServerSockets

El progreso se guarda en `localStorage` del navegador.
